### Información general
***
El proyecto "Mapping" consta de una aplicación web desarrollada en Python y en HTML en la cual se muestra un mapeo mundial, en el que se muestran 2 diferentes capas: la primera que muestra los volcanes en el país 'United States of America'; la ssegunda capa grafica la cantidad de población de los respectivos países iluminandolos de 3 diferentes colores dependiendo del nivel de población existente en el año 2005.
El estatus del proyecto actualmente es finalizado.
